# Optimal-PhiBE

Yuhua Zhu, Yuming Zhang, and Haoyu Zhang.  
**Optimal-PhiBE: A PDE-based Model-free framework for Continuous-time Reinforcement Learning.**  
arXiv:2506.05208, 2025. [https://arxiv.org/abs/2506.05208](https://arxiv.org/abs/2506.05208)

```bibtex
  author={Zhu, Yuhua and Zhang, Yuming and Zhang, Haoyu},

  year={2025}
}
```

## Packages

Use Python 3.10 if possible. Install the required packages with:

```powershell
python -m pip install -r .\Optimal_PhiBE\requirement.txt
```
